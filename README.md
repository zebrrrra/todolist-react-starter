# Todo List X MVC with React
- 此為AC學期3 Todo List之作業，使用React打造的前端應用程式

### You Can Do...
- 新增todo項目
- 編輯修改todo項目
- 刪除todo項目

### 安裝
- 打開終端機 輸入指令 `git clone 'https://github.com/zebrrrra/todolist-react-starter'`
- 進入專案資料夾 `cd todolist-react-starter`
- 安裝所需套件 `npm install`
- 啟動專案 `npm start`
- 在瀏覽器輸入 http://localhost:3000 可瀏覽本專案

### 環境設置
- React 18.2.0
- Node.js 14.16.0
- npm 6.14.11
